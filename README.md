# 724 Events

Le site d'une agence évènementielle.

## Mission

L'application présentait plusieurs bugs et notre mission était de débuguer le site.

### En détail

- Réparation du carrousel en affichant les éléments par date, en ordre décroissant
- Créer un filtrage sur la section "Nos réalisations"
- Corriger l'affichage des mois sur la section "Nos réalisations"
- Réparer le formulaire et y ajouter un message de confirmation

### En plus

- La mise en place de tests unitaires (trois ou plus) et de tests d'intégrations (trois ou plus) n'était pas obligatoire mais suggérée afin d'améliorer la couverture du code.

## Technologies

- HTML
- CSS
- JavaScript
- React

## Process d'installation
### Pre-requis
- NodeJS  >= v16.14.1

### Installation
- `yarn install`

### Lancement de l'application
- `yarn start`

### Tests
- `yarn test`
